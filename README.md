Java es el rey del desarrollo empresarial gracias a su fuerte tipado y su estructura. Este programa simula un backend para gestionar habitaciones de hotel. Utiliza conceptos avanzados como Herencia, Polimorfismo y Encapsulamiento.

Crea un archivo en VS Code llamado SistemaReservas.java

Nota para ejecutar Java en VS Code: Asegúrate de instalar el "Extension Pack for Java" desde la tienda de extensiones de VS Code para que te aparezca el botón de ejecutar ("Run").
